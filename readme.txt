A lightweight plugin that displays icons floating across the screen or a specific area of the webpage. There are 5 icons (Hearts, Trees, Stars, Asteriks, Drops) to choose from, which makes it perfect for sales offer, discount offers, Christmas effects and other special occasions when you want something extra for your webpages. 

== Description ==

Floating Icons enhances the user experience during special occasions or at times when you are offering some sales or discount. It grabs user attention by greeting the floating icons which are moving across the screen.

Perfect for festive effects, sales and discount season, for crazy-bloggers or anyone who is willing to show some effects to greet the user.

[[Check out the Demo](https://www.slickpopup.com/free-wordpress-floating-icons-plugin/?utm_source=floatingicons&utm_medium=wporg&utm_campaign=OmAkSols&utm_term=floatingiconlink)]

*Note:* Basic jQuery should not have any conflicts on your website.

**Features of the plugin:**

* Add effects to your landing pages 
* 5 pre-built icons to chose from
* Ability to use any of the font-awesome icons
* Icons will be created within a range of three color values
* Ability to choose size of the icons (minimum and maximum)
* Ability to set speed of the icons to flow down on the screen

**Other advantages:**

* Easy to use and setup
* Filter to add new icon using filters
* Supports full house of font-awesome icons
* Can be customized to view on a specific element

If you are running a special event on the site then this plugin can add an extra flair to your pages. 

== Installation ==

The quickest method for installing the Slick Popup Lite plugin is:

1. Visit Plugins -> Add New in the WordPress dashboard
2. Search for "Floating Icons"
3. Click "Install Now"
4. Finally click "Activate Plugin"

If you are WordPress fan and prefer to do things manually then follow these instructions:

1. Upload the `floating-icons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit settings from the Dashboard Options page "Floating Icons"

== Frequently Asked Questions ==

= How to show the icons in a specific element? =

Go to Settings and under zone you will option "Specific Element" and there you can add id or class to which you want to show the icons.

= Can I create floating icons on more than one page? =

No, this feature will be available upcoming updates.

= Do Floating icons look the same in every theme? =

We tried to prepare HTML / CSS code to work regardless of the theme, but if you have a problem on your website, please contact us for assistance. Email: poke@slickpopup.com

= What version of WordPress I need? =

Plugin will work in WP 3.2+. However, we recommend using the latest version.

= Is it possible to enable icons only for a selected page? =

Yes, you can do it in Plugin settings, just check for the "Select Page" options in the settings.

= I have a doubt, what should I do? =

For any doubts or queries, please read our F.A.Q section. If it doesn't come out to be helpful, you can contact support at: poke@slickpopup.com
